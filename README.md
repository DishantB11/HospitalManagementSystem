# Hospital Management System

## Overview
A comprehensive Java-based Hospital Management System that streamlines patient registration, doctor management, and appointment scheduling for small to mid-sized healthcare facilities.

## Features
- **Patient Management**: Register, view, and manage patient records
- **Doctor Management**: Maintain doctor profiles and specializations
- **Appointment Scheduling**: Book and manage patient-doctor appointments
- **User Authentication**: Basic user role management (Admin, Doctor, Patient)
- **Reporting**: System summary and status tracking

## Technologies Used
- **Programming Language**: Java
- **IDE**: IntelliJ IDEA
- **Version Control**: Git
- **Build System**: IntelliJ Native
- **Data Storage**: In-Memory Collections (ArrayList)
